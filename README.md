### JSON Persistence
- The application starts with **30+ movie records** stored in `movies.json` located in the movie-watchlist-api folder
- All CRUD operations modify the JSON file on the server
- Data persists across:
  - browser refreshes
  - incognito/private sessions
  - different devices (when backend is hosted)
